# MOVIE

This project consists of a single class (**Filme**) and an enum (**GENERO**).  
The class has the following attributes:  
- title  
- duration  
- genre  

And the methods:  
- getTitle  
- setTitle  
- getDuration  
- setDuration  
- getGenre  
- setGenre  

The enum defines the genres:  
- HORROR  
- COMEDY  
- ROMANCE
